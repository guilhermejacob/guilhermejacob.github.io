Estimando a linha de riqueza dos 1% mais ricos usando pesquisas amostrais

 Feb 2, 2021
Nos últimos dias, ocorre uma discussão interessante sobre quem são os “ricos”: o 1% mais rico da população brasileira. Uma das perguntas mais interessantes é a partir de qual renda é considerado rico. Esse número mágico é difícil de estimar usando pesquisas amostrais por três razões principais: não-resposta, erros de medida e erro amostral. Existem algumas indicações de que há maior não-resposta entre os domicílios mais ricos. Isso ocorre tanto como não-resposta de unidade, quanto não resposta de item, gerando problemas para a análise destes dados (Yan, Curtin e Jans, 2010).
